    Help & Information

    This application accepts PDF files containing Form 31 bankruptcy documents. The PDF files can be:

    • Fillable PDF forms that have been completed electronically
    • Scanned documents of completed Form 31 (OCR will be attempted)

    Maximum file size: 10MB per file

    The generated XML files comply with the ISED (Innovation, Science and Economic Development) Canada e-filing system schema for bankruptcy Form 31.

    XML files include all necessary elements and attributes required for successful e-filing with the Canadian government system.

    Common errors include:

    • Password-protected or encrypted PDF files cannot be processed
    • PDFs with missing required fields may generate incomplete XML
    • Scanned documents with poor quality may result in extraction errors

    All conversion results provide detailed error messages to help troubleshoot issues.
